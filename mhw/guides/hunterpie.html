HunterPie Overlay & Rich Presence

HunterPie is a modern and simple to use overlay with support for Discord Rich Presence for Monster Hunter: World.

The things you can do with HunterPie include the following:

  1. Get a brand new player info bar with more details
  2. Damage & DPS Parsing
  3. Detailed Class Helpers
  4. Plugins/Add-Ons that give extra features
  5. Upload Current Loadout In-game to HoneyHunter
  6. Discord Rich Presence, which allows one to see HR/MR, Weapon Currently Equipped, What Locale your in, What your hunting with health % remaining and quest time. All this can be seen when clicking on a player in discord.

Plugin Setup

♦ Discord Webhook Plugin

  1. First Launch the Game after installing the plugins.
  2. Then go to HunterPie -> Plugins -> Click on DiscordWebhook and on the right panel, look for Plugin Directory.
  3. Open the folder and look for Config.json. Open the file in Notepad or Notepad++
  4. Now go to Legacy Nation Discord. Look for the MHW Channel in the Channel list and click on the ⚙️ Icon.
  5. Look for Webhooks in Integrations and create a new one and name it after your character name in MHW.
  6. Click the Copy URL Button and then go to the Config.json file you opened and paste in your Discord Webhook Link and save the file.
            
                shift+ctrl+p | Sends DPS data from latest combat to Discord channel.
                shift+ctrl+g | Sends your current build to Discord channel, the link is to Honey Hunters World, but it's shortened using Tinyurl.

♦ Discord Helper Plugin

  1. First Launch the Game after installing the plugins.
  2. Then go to HunterPie -> Plugins -> Click on Discord Helper and on the right panel, look for Plugin Directory.
  3. Open the folder and look for Config.json. Open the file in Notepad or Notepad++
  4. Copy the Unique ID in the " " and run the command in the MHW Channel: $add PASTEYOURID - to add it to the bot so you can use the commands.
  5. Remove 1 "S" from the URL address in the config and save the file.
  6. It should look like this:
            "uri": "ws://server-mhwdiscordhelper.herokuapp.com/"
